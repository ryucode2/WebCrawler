# WebCrowler

A lightweight web crawler written in C.

## Features

- URL crawling
- Queue management
- Graph-based URL tracking
- Duplicate URL detection
- robots.txt handling

## Build

Requirements:

- GCC
- Make

Build:

```bash
make

Run:
./bin/crawler


Project Structure:

- src/       Source code
- include/   Header files
- build/     Build output
- bin/       Executable
